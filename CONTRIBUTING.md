# Contributing to Intivolution AI

Thank you for your interest in contributing! This project was built as a Final Year Project and is now open to community contributions.

## 🐛 Reporting Bugs

1. Check if the issue already exists in [Issues](../../issues)
2. If not, open a new issue with:
   - A clear title
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots (if applicable)

## 💡 Suggesting Features

Open an issue with the `enhancement` label and describe:
- The problem you want to solve
- Your proposed solution
- Any alternatives you considered

## 🔧 Making Changes

1. **Fork** the repository
2. **Clone** your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/intivolution-ai.git
   ```
3. **Create a branch**:
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Set up the project** (follow [README.md](README.md))
5. **Make your changes** and test them
6. **Commit** with a meaningful message:
   ```bash
   git commit -m "feat: add skill comparison chart"
   ```
7. **Push** your branch and open a Pull Request

## 📋 Code Style

- Use existing patterns in the codebase (Next.js App Router, Tailwind CSS)
- Keep components small and focused
- Add comments for complex logic
- Test your changes before submitting
